Разработай полноценную реферальную систему для моего Telegram-бота. Продумай ее как коммерческий продукт, а не как простое приглашение друзей.

Основные требования:

## Общая логика

Каждый пользователь автоматически получает уникальную реферальную ссылку.

Например:

https://t.me/MyBot?start=ref_xxxxxxxxx

или deep link для Mini App.

При переходе по ссылке пользователь закрепляется за пригласившим.

Реферал засчитывается только если новый пользователь:

• впервые зарегистрировался;
• полностью подключил аккаунт;
• завершил первоначальную настройку;
• не является дубликатом.

До выполнения этих условий реферал имеет статус "Ожидает активации".

Добавь красивые карточки, прогресс-бары и современный UI.

## Награды

Продумай гибкую систему наград.

Например:

1 приглашенный
→ +3 дня Premium

3 приглашенных
→ +7 дней

10 приглашенных
→ +1 месяц

25 приглашенных
→ эксклюзивный значок

50 приглашенных
→ уникальный профиль

100 приглашенных
→ пожизненная скидка

Сделай так, чтобы награды легко изменялись через админ панель.

## Защита от накрутки

Продумай полноценную антифрод-систему.

Например:

• нельзя приглашать самого себя;
• нельзя использовать один Telegram ID несколько раз;
• нельзя использовать один аккаунт несколько раз;
• проверять возраст аккаунта;
• проверять завершение регистрации;
• учитывать только активных пользователей;
• предотвращать циклические приглашения;
• логировать подозрительные действия;
• возможность вручную отменить начисления через админ-панель.
